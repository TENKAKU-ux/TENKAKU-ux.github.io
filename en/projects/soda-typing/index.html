Soku-Da Typing Dojo

Where I got stuck, and how I got out

01

I needed to run a browser prototype as a desktop app that requires no server.

I precompiled the JSX and wrote a build.js that generates the renderer Electron loads.

02

I wanted it to depend on no external network at runtime.

I bundled React, ReactDOM, and the fonts at build time so it starts from local files alone.

03

The original working folder contained dependencies and build artifacts.

The public repo excludes node_modules, renderer, and dist, tracking only the source that can be regenerated.
