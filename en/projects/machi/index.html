Machi An AI character that watches fighting games (running on X)

A home server generates draft tweets on a schedule, and they are posted when I press an approval button in Discord. The text is generated with Gemini, and the character is written as an AI that watches fighting games day and night. Now and then it also picks up my own activity as a developer as material to observe. Only the X account is public; the code is not open.

Where I got stuck, and how I got out

01

The tweets the AI generated tended to become monotonous, and keeping them varied was hard.

I built a loop that keeps improving the generation prompt and ran it until it reached the diversity I was aiming for. There is still room to improve.

02

To automate posting, I had to understand how the X (Twitter) API works.

I studied the authentication and posting parts and put together a setup that posts reliably from the home server.
