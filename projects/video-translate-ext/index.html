動画リアルタイム翻訳 ブラウザ内で動画をその場翻訳（Chrome 拡張）

YouTube・X をはじめ、Chrome で再生できる動画なら幅広く翻訳できます。字幕がある動画は画面に出ている字幕を読み取って Chrome 内蔵の翻訳（オンデバイス）で訳し、字幕が無い動画はタブの音声をローカルの Whisper で文字起こししてから訳します。英語や中国語などの動画を日本語に翻訳でき、音声や文字は端末の外へ送らず、外部 API も使いません。サイドパネルに原文と訳が並んで流れ、翻訳した記録は保存して Markdown（.md）ファイルに書き出せます。

詰まった点と抜け方

01

サイドパネルからタブの音声をどう取り込むか。権限を付けても、音声を掴む標準の仕組みが動かず詰まった。

サイドパネル内のボタンを押した瞬間だけ許可が下りることに気づき、そのクリックを起点に音声つきの画面共有を開始する形にした。

02

YouTube の字幕データを直接取りに行くと、中身が空で返ってくるようになっていた。

画面に表示されている字幕を読み取る方式に切り替え、視聴者が字幕表示を ON にする前提で安定して拾えるようにした。

03

拡張の常駐部分が数十秒で勝手に止まり、覚えていたはずの対象タブを忘れてしまう。

セッション用のストレージに状態を退避し、処理の前に必ず読み直してから続けるようにした。
