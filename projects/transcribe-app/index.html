もじおこし 文字起こし＋要約コピー補助

動画・音声をブラウザから投入すると、ローカルの Whisper（small・CPU）が文字起こしし、進捗をリアルタイム表示します。完了後は講義ノート・要点箇条書き・試験対策の3プリセットから選び、プロンプト＋全文を一発コピーして ChatGPT に貼るだけ。履歴・txt/srt ダウンロード付き。サーバーは自宅 PC、外部 API は使いません。

詰まった点と抜け方

01

家庭内 LAN の HTTP ページでは navigator.clipboard が使えず、肝心のコピーが失敗する。

execCommand によるフォールバックを用意し、それも失敗したら全選択済みテキストエリアを表示して、必ずコピー手段が残るようにした。

02

CPU での文字起こしは長時間かかるのに、標準では進捗が分からない。

ffprobe で得た総尺と Whisper が逐次出力するセグメントのタイムスタンプから進捗率を推定し、認識中のテキストもリアルタイムで流すようにした。

03

長い文字起こし全文を ChatGPT にどう渡すか迷った。

基本はプリセットプロンプト＋全文の一発コピーとし、長文のときだけ分割コピーを選べる二段構えにした。
